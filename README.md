# Heat Pump Modeling Tool (TESPy Case Study)

This repository contains an early-stage implementation of a heat pump modeling tool built using the TESPy library, developed as part of a case assignment.

The goal of this project is to simulate and analyze a heat pump system based on provided case data using thermodynamic process modeling.


## Project Status

 **Work in progress / not fully functional**

The current implementation is not yet fully operational. While the project structure and initial components are in place, the TESPy model can't be successfully executed due to implementation and compatibility issues.

Key challenges encountered:
- Limited prior experience with TESPy
- Differences between available documentation/examples and the current TESPy version
- Incomplete or outdated reference material for certain modeling approaches

This repository reflects the current state of development and experimentation.
